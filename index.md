As 3PP contests become more relevant, a need arises for analysis tools.

This page lets you enter certain preference-flow assumptions and get a
graph back.

You can mouse over the dots to get a tool-tip with details of 3PPs and
estimated winner and margin, based on the assumed preference flows.

Note that form input is in percentages.

This graphing tool was created at the instigation of [Ben Messenger](https://twitter.com/BenMessenger8/),
who has published some [further analysis.](https://www.3partypreferred.net/3partypreferred/what-is-a-three-party-preferred)

Check out the source code [on GitHub.](https://github.com/alexjago/3pp-visualiser)
